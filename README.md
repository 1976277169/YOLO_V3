YOLO_v3
=
YOLO_v3 implemented with tensorflow <br>

- [x] Group Normalization<br>
- [x] Focal loss<br>
- [x] Soft-NMS(no useful)<br>
- [x] data augmentation<br>
- [x] multi-scale training<br>
- [ ] Single-Shot Object Detection with Enriched Semantics<br>
- [ ] SNIP<br>
